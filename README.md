# People Counter with YOLOv8

This project uses YOLOv8 and Streamlit to detect and count people in images.

## Features
- Upload image
- Detect people
- Count number of persons
- Display bounding boxes

## Installation

```bash
pip install -r requirements.txt
```

## Run the App

```bash
streamlit run app.py
```

## Model
Uses YOLOv8m model from Ultralytics.
